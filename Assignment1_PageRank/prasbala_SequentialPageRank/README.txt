Files:

SequentialPageRank.java 	- the source code to be compiled
pagerank.input			- input file with 10 nodes
pagerank.input.1000.urls.26	- input file with 1000 nodes
pagerank.output			- output file generated by the program
Report.pdf			- report for project

Execution method:

Step 1: compile the java code with following command:

javac SequentialPageRank.java

Step 2: execute with the parameters as follows:

java SequentialPageRank <input file> <output file> <iterations> <damping factor>


example:

java SequentialPageRank pagerank.input pagerank.output 10 0.85
